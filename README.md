AI FAQ Chatbot

This project is a simple AI chatbot built using Python and Hugging Face Transformers.

Features
- Uses an open-source LLM (TinyLlama)
- Answers user questions interactively
- Demonstrates basic NLP and LLM integration

Technologies
- Python
- Transformers library
- TinyLlama chat model

How to Run
1. Install dependencies
2. Run the Python script
3. Ask questions in the terminal
